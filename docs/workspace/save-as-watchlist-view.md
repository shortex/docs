# Save as Watchlist view

[**Learn more about Watchlists**](/category/watchlist/)

This is so you can fully customize the view that is iterated over when loading a watchlist.

```Workspaces -> Save as```


**Things you need**
- Not more than one view in your workspace
    - These views need to be named as alawys
